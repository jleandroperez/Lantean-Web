<img class="alignleft size-medium wp-image-681" style="margin-left: 5px; margin-right: 5px;" alt="MacbookPro" src="/wp-content/uploads/2013/01/MacbookPro.jpg" width="149" height="110" />So... you own an iMac.. maybe a Macbook Pro, or maybe you're one of the lucky owners of a Mac Pro. You've been watching ads for the last ten years... you bought it all, and you feel safe just because it's not a windows... right?.

Well... things are not the way they were a couple years ago. As Apple gained more and more marketshare, virus developers turned their eyes on our belived OSX system. Since Java virtual machine has been found to be buggy... several viruses that affected Mac have been caught in the wild.

What can we do?. Well.. there are a couple things you could do....

<strong>1. Enable Mac Gatekeeper</strong>

What is the Mac Gatekeeper?. Well, the concept is simple. We just close the door to malicious software, by only installing software from trusted sources. Ever wondered why there are no viruses for iPhone?. Well... it's because iOS ecosystem is a walled garden. By enabling the Gatekeeper, you'd be replicating the same safe environment as iOS has.

In order to enable the GateKeeper, you need to go to <em>"System Preferences &gt;&gt; Security &amp; Privacy"</em> (in Mountain Lion). You will get this screen:

<a href="/antivirus-for-mac/screen-shot-2013-01-23-at-11-36-08-pm/" rel="attachment wp-att-648"><img class="alignnone size-full wp-image-648" alt="Screen Shot 2013-01-23 at 11.36.08 PM" src="/wp-content/uploads/2013/01/Screen-Shot-2013-01-23-at-11.36.08-PM.png" width="663" height="513" /></a>

See where it says 'Allow applications downloaded from'?. Make sure you check 'Mac App Store' (or Mac App Store + identified devs). 'Anywhere' is not recommended.

That way you're computer will install software ONLY from trusted sources.

<strong>2. Install Mac Antivirus Software</strong>

Yes... installing an antivirus will impact, slightly, on your performance. But at the very least, you'll get to feel safe again. It's just a feeling, bear that in mind, always!.

What antivirus software can you use?. Well, you could try <a title="AVG" href="http://free.avg.com/es-es/linkscanner-mac">AVG for Mac</a> (which is free!), <a title="ESET for Mac" href="http://www.eset.com/us/home/products/antivirus-for-mac/">ESET for Mac</a> (paid, but one of the best), and <a href="http://www.symantec.com/security_response/definitions/download/detail.jsp?gid=nmc">Norton Antivirus for Mac</a>.

<strong>3. Install a Firewall</strong>

One of the things that make me freak out the most... is not knowing what applications are openning connections with remote servers. So... if you're paranoid like me, you could simply install a <a title="Little Snitch" href="http://www.obdev.at/products/littlesnitch/index.html" rel="nofollow">firewall</a>, and validate, process by process, if they really need to access internet.

Hopefully, with this extra software you'll be able to stay away from trouble. Again... hopefully!.

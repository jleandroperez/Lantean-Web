If you're reading this post, you're probably aware of the dangers of Flash Player. And you've read, most probably, about the last couple of viruses / penetrations to Facebook and Google.

Steve Jobs hated Adobe Flash...  personally, i say he had a very good reason. If wasn't, probably, just because Adobe refused (at first) to publish its suite for Mac. He knew that it was troublesome... so... let's proceed removing that junk from our system!.

<h5><strong>First: Download the Uninstaller</strong></h5>

If you're running Snow Leopard (or newer), you're gonna need to download the <a href="http://fpdownload.macromedia.com/get/flashplayer/current/support/uninstall_flash_player_osx.dmg">Flash Player Uninstaller</a>. However, if you're running an older version (Leopard and Tiger), please, download <a href="http://fpdownload.macromedia.com/get/flashplayer/current/support/uninstall_flash_player_osx_10.4.dmg">this uninstaller</a>.

You're gonna need to double click on the DMG file, and click on the "Adobe Flash Player Uninstaller" application. You should get the following window onscreen:

<a href="/wp-content/uploads/2013/02/uninstall-flash-player.png"><img class="aligncenter size-full wp-image-1149" alt="uninstall-flash-player" src="/wp-content/uploads/2013/02/uninstall-flash-player.png" width="424" height="270" /></a>

<h5>

<strong>Second: Proceed uninstalling</strong></h5>
This is pretty straightforward. Simply hit the 'Uninstall' button, and it will all be over, soon enough. You will be required to close all of your browsers first. Once it's all over, you should see the following screen:

<a href="/wp-content/uploads/2013/02/remove-flash-player-from-mac.png"><img class="aligncenter size-full wp-image-1150" alt="remove-flash-player-from-mac" src="/wp-content/uploads/2013/02/remove-flash-player-from-mac.png" width="420" height="270" /></a>
Congratulations! welcome to the future!. HTML 5 is becoming the web standard... and we certainly don't need Adobe to run anything on a browser.

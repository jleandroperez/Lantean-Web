<p>If you're running <strong>Windows 10</strong> on a <strong>Macbook Pro 2013 (Early)</strong>, and constantly stumble upon an awesome <strong>Machine Check Error</strong> screen whenever you reboot, you shoooould try:</p>

1. Open the device manager
2. Locate you GeForce GT 650M
3. Uninstall the 2017 Driver
4. Reboot
5. Let Windows install the system's driver
